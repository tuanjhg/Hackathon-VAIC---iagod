# Banner images

Thả ảnh của bạn (đã có bản quyền) vào thư mục này theo đúng **tên file** bên dưới.
Chưa có ảnh thì component vẫn hiển thị placeholder gradient đẹp — có ảnh là tự thay, không cần sửa code.

## Hero slider (băng chuyền chính)
Tỉ lệ ~ **16:7** (gợi ý 1280×560 hoặc 1440×630), JPG/PNG/WebP.
- `hero-1.jpg`
- `hero-2.jpg`
- `hero-3.jpg`

## Banner phụ (cột phải cạnh hero)
Tỉ lệ ~ **16:10** (gợi ý 320×200), 2 ảnh.
- `side-1.jpg`
- `side-2.jpg`

## Gian hàng ưu đãi (lưới 4 ảnh dọc)
Tỉ lệ **3:4** (gợi ý 480×640).
- `promo-1.jpg`
- `promo-2.jpg`
- `promo-3.jpg`
- `promo-4.jpg`

## Banner đôi (2 ảnh ngang)
Tỉ lệ **5:2** (gợi ý 800×320).
- `sub-1.jpg`
- `sub-2.jpg`

> Muốn đổi liên kết khi bấm banner, sửa `href` trong `apps/web/lib/banners.ts`.
> Nếu dùng đuôi khác `.jpg` (vd `.webp`), đổi tên tương ứng trong `banners.ts`.
